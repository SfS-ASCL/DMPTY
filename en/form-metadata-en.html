Documentation and Metadata

What information is necessary so that your research data is readable and interpretable in the future? Please describe how you document the primary research data. Here, the data should be described as specifically as possible in plain text. For measured data, please give for example the definitions of the variables; for annotated corpora, please include a glossary of all annotations as well as the annotation hand book.

After the plain text description, please use a metadata format which is as standard as possible to describe your data. For this purpose, please state the metadata format (the “profile”) used for each data type. In principle, the Component Metadata Infrastructure (CMDI) is used to describe the metadata. If necessary, existing CMDI profiles can be adapted according to project needs.
