Description of the Primary Research Data

In this section, please give a short description of your research data. Here, describe only the data that will be created over the course of the project. If stock data are to be used in your project, please describe them later in the form.
Please consider that the choice of data format and processing software have an impact on data archiving and reuse. Whenever possible, open formats and Open-Source Software should be used.

Repositories generally accept only certain data types that must be in long-lasting and simple to process formats. The CLARIN-D user handbook describes which data types and formats are supported, see S. 56 ff and section 4.3. Formats not included in these sections must be specified in the Data Management Plan.

If you plan to access stock data in your project, please give a short description of the stock data and their underlying license requirements in the following three fields. You should have the rights to reuse these data and work derived from the data should be able to be made accessible.
